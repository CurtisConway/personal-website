Projects

Some of my work.

Musora Center is the GUI for a simple but elegant content, user, and e-commerce management solution. All of Musora's applications utilize Musora Center in some way.

A single page web application, Musora Center is built with VueJS - utilizing the Vue Router for routing, Vuex for state management, and the Vuetify component library.

Vuesora is a one-stop shop custom front-end framework. Vuesora powers the front-end for all of Musora's web applications - Drumeo, Guitareo, Pianote, and Recordeo.

Vuesora consists of 2 main modules

  • A complete CSS framework - with a 12 point grid system based off of the flex-box spec. Vuesora also has an intuitive naming convention for all sorts of modifiers, from typography to colors, borders and dozens of styled components - Vuesora allows views to be built with minimal specific CSS.

  • A custom Vue Component library. Any of the application's components that require complex DOM manipulation have their own custom Vue component. Vuesora has dozens of custom components, including but not limited to:

    • A dynamic and highly configurable catalogue system for browsing content
    • HTML5 Video Player
    • Image Cropper
    • WYSIWYG Text Editor

A quick look into my work.

Musora Center is the GUI for a simple but elegant content, user, and e-commerce managment solution. All of Musora's applications utilise Musora Center in some way.

A single page web application, Musora Center is built with VueJS - utlizing the Vue Router for routing, Vuex for state mangement, and the Vuetify component library, conforming the application to the Material Design specification.
